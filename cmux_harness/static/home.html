1. Configure cmux

cmux needs Automation mode enabled so the harness can read sessions and send approved input.

1
Open cmux Settings
In the Mac app, open Settings, then choose Automation.
2
Enable the socket API
Set socket control to automation. Restart cmux if this page still reports that the socket is missing.
3
Keep this server running
The iOS app talks to this Python server, not directly to cmux.

2. Connect from iPhone

Use LAN when you are on the same Wi-Fi. Add Tailscale when you want the iOS app to work away from home or office.

Searching local network and Tailscale
Checking localhost, LAN, Bonjour, and Tailscale URLs...
This is optional. Install Tailscale on the Mac and iPhone, sign into the same tailnet, then use the detected URL or enter the Mac device name shown by Tailscale. Tip: set a stable Tailscale machine name like cmux-mac and turn off OS-hostname auto-generation so this URL does not change when your Mac hostname changes.
